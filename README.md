# URJA '23 (Official Website)

**A national-level sports fest celebrating legacy, competition, and collective spirit.**

URJA ’23 is the official sports festival of Thapar Institute of Engineering and Technology, designed as a high-energy, digitally powered platform that unifies athletes, teams, and spectators. It blends competitive excellence with immersive web experiences, capturing the scale and intensity of a modern sports fest.

---

## Core Highlights

- **Event Platform:** Centralized hub for all sports events, schedules, and participation  
- **Dynamic UI:** Smooth, responsive interface with engaging visual transitions  
- **Registration Flow:** Streamlined participant onboarding and event sign-ups  
- **Live Updates:** Real-time announcements, fixtures, and results tracking  
- **Data Handling:** Efficient collection and management of participant data  
- **Form Systems:** Robust input handling with validation and structured workflows  
- **Cloud Backend:** Firebase-powered data storage and real-time syncing  
- **Email Integration:** Automated communication using EmailJS  
- **Export Tools:** CSV-based data extraction for organizers and analytics  
- **Viewport Animations:** Scroll-based interactions enhancing user engagement  

---

## Tech Stack

**Core:** React 18, JavaScript, React Scripts  
**Backend & Data:** Firebase  
**Networking:** Axios  
**Forms & Input:** React Dropdown, React Hook Utilities  
**Integrations:** EmailJS  
**Data Tools:** React CSV, JSON-to-CSV  
**UI & Interaction:** React Icons, Intersection Observer, Viewport Animations  

---

## Vision

URJA ’23 is built to digitally amplify the spirit of sport—where competition meets community, and every interaction reflects the energy, unity, and legacy of the fest.
